# # Cybersecurity Toolkit

A collection of beginner-friendly cybersecurity tools built using Python.
This project demonstrates core security concepts such as password security, cryptography, network scanning, packet analysis, and basic web security checks.

## Features

* Password Strength Checker
* Hash Generator
* Dictionary Password Cracker
* Brute Force Simulation Tool
* Port Scanner
* Packet Sniffer
* Web Vulnerability Scanner

## Technologies Used

* Python
* Socket Programming
* Requests Library
* Scapy

## How to Run

Clone the repository:

git clone https://github.com/adityadutta6069/cybersecurity-toolkit.git

Navigate to the project folder:

cd cybersecurity-toolkit

Run the toolkit:

python toolkit.py
